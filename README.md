# ascii-art-web
To run web server 👉 go run main.go 
It will be hosted on port :9800, it will display in terminal to also let you know
Enter LocalHost 9800 in web browser to access Ascii Art Web GUI
Type in text with chosen banner, and it will display the results
Enjoy!
